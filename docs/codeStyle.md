

### code style


#### css

```
css文件起名以下划线开头，驼峰命名如：
_login.less

css样式写法规范，缩进四个空格，class与大括号之间要有一个空格，
属性冒号后与值也需要一个空格
如：
.login {
    color: #fff;
}

```

#### react 文件命名
react组件也是用驼峰命名，不过首字母大写，
```
/app/view目录下的文件以view结尾如:
LoginView.js

/app/components目录下的文件不需要以view结尾
```

#### js 文件命名

所有的js也遵循峰驼命名首字母不大写
